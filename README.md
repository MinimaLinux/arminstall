# minimalinux installer
![arm64 supporters](https://img.shields.io/badge/arm64-supporters-pink)
quick installer for minimalinux.  
## installation
easy. to install, download debian [here](https://www.debian.org/distrib/netinst).  
if you already use ubuntu, you don't need to do this step.  
if you downloaded debian, install it.  
okay, now run this to get our script:  
```
curl https://tinyurl.com/MinimaLinux | bash
```
enjoy.
